Custom ipay88 gateway plugin for woocommerce.

== Description ==

This plugin created just to help online entrepreneurs using wordpress out there to enable ipay88 to their site.
This plugin is created on `13 April 2016` By: [AiFAiZ](https://github.com/aifaiz) .
We hardly find people do this and it is hard to google. Hope this help!

== Installation ==

1. Install plugin
2. Activate plugin
3. Go to pages and create new page.
4. The new page let it be blank and choose "Payment" template
5. Go to your woocommerce checkout settings
6. Happy selling!


== Frequently Asked Questions ==

= Do you include the Merchant ID together? =

We do not provide the Merchant ID, You will need to register with ipay88 as Merchant to obtain the Merchant ID.

= Are you associated with ipay88? =

No. We are just a web developer that sees an opportunity to help ease business process for startup.
